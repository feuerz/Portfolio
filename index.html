Soy un desarrollador web apasionado por la tecnología y la creación de soluciones digitales. Tengo experiencia tanto en el desarrollo frontend como backend, creando soluciones digitales limpias, eficientes y escalables.

Turtle Race Game

Un proyecto interactivo desarrollado con HTML, CSS y JavaScript, en el que cuatro tortugas compiten en una carrera hasta la meta. Este ejercicio, además de ser entretenido, está orientado a demostrar el uso de lógica de programación asincrónica en JavaScript mediante setTimeout y setInterval, control de animaciones y manejo de eventos. Se trata de un ejemplo sencillo pero didáctico que refleja la capacidad de integrar dinámica visual con conceptos fundamentales de programación.

Tecnologías: JavaScript, HTML, CSS, Git
