Encinar & Manzano – Web + Panel de Control

Desarrollo de una plataforma completa compuesta por una web corporativa totalmente dinámica y un panel de control interno. El cliente puede gestionar servicios, proyectos, imágenes del carrusel y todo el contenido visible en el frontend. Además, el panel interno permite administrar usuarios y roles, gestionar el CRM, controlar fichajes y realizar tareas de facturación, ofreciendo una solución centralizada y flexible para la operativa diaria de la empresa.

Tecnologías: JQuery, PHP, MySQL, HTML, CSS

Turtle Race Game

Proyecto web interactivo desarrollado con HTML, CSS y JavaScript, en el que cuatro tortugas compiten en una carrera mediante animaciones controladas por lógica asincrónica. El ejercicio demuestra el uso de setTimeout y setInterval para gestionar tiempos y actualizaciones periódicas, así como la manipulación del DOM, el control de eventos y la sincronización de movimientos en pantalla.

La visualización en dispositivos móviles aún no está completamente optimizada, ya que se encuentra en proceso de implementación de reglas responsivas y ajustes de layout para mejorar la experiencia en pantallas reducidas.

Tecnologías: JavaScript, HTML, CSS, Git

ToT - Web Minijobs Locales

Proyecto Final de DAW: plataforma web que conecta a usuarios para realizar miniJobs locales. Permite publicar tareas, gestionar perfiles, enviar valoraciones y manejar imágenes de forma segura. Desarrollada con PHP, MySQL, JavaScript, HTML y CSS, y adaptada con un diseño totalmente responsive para móvil, tablet y escritorio.
